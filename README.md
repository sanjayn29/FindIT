# Find It All

A comprehensive, all-in-one search platform built with modern web technologies. Find It All helps users discover medical professionals, job opportunities, educational resources, housing rentals, products, and transportation services - all in one place.

## Features

- **Medical Services**: Find doctors, hospitals, and healthcare providers
- **Job Search**: Browse and search for job opportunities
- **Education**: Connect with teachers and educational institutions
- **Housing Rentals**: Search for rental properties
- **Online Sales**: Browse products and services
- **Transportation**: Book rides and find transport options
- **User Profiles**: Manage your personal information and preferences
- **Saved Items**: Bookmark and save your favorite listings

## Technologies Used

This project is built with:

- **Vite** - Fast build tool and development server
- **TypeScript** - Type-safe JavaScript
- **React** - UI component library
- **shadcn/ui** - Beautiful, accessible component library
- **Tailwind CSS** - Utility-first CSS framework
- **Supabase** - Backend and authentication
- **React Query** - Data fetching and state management
